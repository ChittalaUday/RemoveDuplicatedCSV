Process Explanation / ప్రక్రియ వివరణ

1. ఫైల్ ఎంపిక:

మీ Excel లేదా CSV ఫైల్‌ని డ్రాగ్ చేసి డ్రాప్ చేయండి లేదా "Browse Files" బటన్‌ని క్లిక్ చేసి ఎంచుకోండి.

2. ప్రాసెసింగ్ దశలు:

  • డూప్లికేట్ ఎంట్రీలు తొలగించబడతాయి (కస్టమర్ పేరు మరియు సీరియల్ నంబర్ ఆధారంగా)
  • 'z1', 'z', 'kap', 'ang' కలిగి ఉన్న ఎంట్రీలు తొలగించబడతాయి
  • తేదీ ప్రకారం క్రమంలో అమర్చబడతాయి
  • అపార్ట్‌మెంట్ రకం ప్రకారం వర్గీకరించబడతాయి (A, B, C)

3. ఫలితాలు:

  • ప్రాసెస్ చేసిన డేటా వివిధ షీట్‌లలో విభజించబడుతుంది:
  • - అన్ని రికార్డులు
  • - A అపార్ట్‌మెంట్ రికార్డులు
  • - B అపార్ట్‌మెంట్ రికార్డులు
  • - C అపార్ట్‌మెంట్ రికార్డులు
  • - ఇతర రికార్డులు

4. డౌన్‌లోడ్:

ప్రాసెస్ పూర్తయిన తర్వాత, "Download Processed File" బటన్‌ని క్లిక్ చేసి ప్రాసెస్ చేసిన ఫైల్‌ని డౌన్‌లోడ్ చేసుకోవచ్చు.

గమనిక: మీ ఫైల్‌లో పైన పేర్కొన్న అన్ని కాలమ్‌లు ఉండేలా చూసుకోండి. లేకపోతే ప్రాసెసింగ్ విఫలం కావచ్చు.

1. File Selection:

Drag and drop your Excel or CSV file, or click the "Browse Files" button to select it.

2. Processing Steps:

  • Removes duplicate entries (based on Customer Name and Serial Number)
  • Filters out entries containing 'z1', 'z', 'kap', or 'ang'
  • Sorts entries by date in ascending order
  • Categorizes by apartment type (A, B, C)

3. Results:

  • Processed data is divided into different sheets:
  • - All Records
  • - A Apartment Records
  • - B Apartment Records
  • - C Apartment Records
  • - Other Records

4. Download:

After processing is complete, click the "Download Processed File" button to download the processed file.

Note: Please ensure your file contains all the columns mentioned above. Processing may fail if required columns are missing.
